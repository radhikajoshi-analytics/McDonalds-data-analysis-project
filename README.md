# 🍔 McDonald’s Sales Analysis Dashboard

## 📖 Overview
This project is an Excel-based sales analysis dashboard created to analyze McDonald’s business performance, revenue trends, and customer purchasing behavior. The dashboard provides meaningful insights through interactive visualizations and reports.

---

## 🎯 Objectives
- Analyze sales performance
- Track revenue trends
- Identify top-selling products
- Compare category-wise sales
- Understand customer behavior

---

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Power Query
- Data Cleaning
- Data Visualization

---

## 📊 Dashboard Features
- Interactive Dashboard
- KPI Tracking
- Sales Trend Analysis
- Revenue Insights
- Category-wise Performance Reports

---

## 🔍 Key Insights
- Top-performing products were identified
- Monthly sales and revenue trends were analyzed
- Customer ordering patterns were tracked
- Business insights were generated using visual reports
